## 🔧 Features Explained

- **God Mode:** Makes your character invincible to damage. Enemies can still attack, but your health will not drop.
- **Infinite Health:** Keep your health bar full at all times.
- **Infinite Amrita:** Amrita is the in-game resource used for leveling and skills. This option prevents your Amrita from running out.
- **One-Hit Kill:** Enemies die after a single attack. Useful to speed through difficult parts.

These features change values in memory while the game runs. The trainer only works when Nioh 3 is open and active.

## 🖥️ How It Works

The trainer uses a technique called memory hacking. It locates the parts of the game data stored in your computer’s memory and changes them to give you the effects listed above.

No game files are modified on disk. Changes only last while the game and trainer are running.

## ❓ Troubleshooting

- If the trainer fails to detect the game, check that Nioh 3 is running before starting the trainer.
- Run both the trainer and the game as the same user. Avoid launching the trainer as Administrator unless necessary.
- Some antivirus programs may flag the trainer. If this happens, temporarily disable your antivirus or add an exception for the trainer folder.
- If you get crashes or freezes, close the trainer and game, then restart both.
- Try downloading the latest trainer version because updates fix bugs and improve compatibility.
